## Features
- Vue 3
- One page layout 
- FireBase connection



## Project setup
```
npm install
```


### Rourter latest version
```
npm install --save vue-router@next
```


### Vuex latest version
```
npm install --save vuex@next
```


### Lints and fixes files
```
npm run lint
```


### Compiles and hot-reloads for development
```
npm run serve
```
